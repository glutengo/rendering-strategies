# Rendering Strategies React Blog

This project was bootstrapped with [Create React App](https://github.com/facebook/create-react-app).

SR Port: `3000`  
SSR Port: `3001`

## Dependencies

`npm install`

## Build

### CSR

`npm run build:csr`

### SSR

`npm run build:ssr`

## Startup

### CSR

`npm run serve:csr`

### SSR

`npm run serve:ssr`

## Development

### CSR

`npm run dev:csr`

### SSR

`npm run dev:ssr`

## Deployment

### CSR

`npm run docker:update:csr`

### SSR

`npm run docker:update:ssr`
