# Rendering Strategies Vue Blog

This project was bootstrapped with [Vue.js CLI](https://cli.vuejs.org).

SR Port: `5001`  
SSR Port: `5002`

## Dependencies

`npm install`

## Build

### CSR

`npm run build:csr`

### SSR

`npm run build:ssr`

## Startup

### CSR

`npm run serve:csr`

### SSR

`npm run serve:ssr`

## Development

### CSR

`npm run dev:csr`

### SSR

`npm run dev:ssr`

## Deployment

### CSR

`npm run docker:update:csr`

### SSR

`npm run docker:update:ssr`
